Projects

Home Network

Inspired by Cumberland's IT department, I set up my own home server on a tight $50 budget. I assembled the components, installed Proxmox, configured Pfsense as my firewall, and deployed Pi-Hole as my DNS server. Subnetting my network allowed me to create a secure environment for running virtual machines. To enhance connectivity, I wired Ethernet cabling throughout my house for better wireless access points.

Security System

Following the creation of my VLAN, I acquired an analog DVR along with cameras. I then ran the camera wiring through the attic to the exterior of my property and connected the DVR to the network. This setup enabled remote access to the security system via Wi-Fi. Additionally, I established a local DNS entry, allowing me to conveniently access the camera network by simply typing "cameras.arpa" into my browser.
